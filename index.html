About LIvRE

LIvRE is a free, open-source, multi-platform Content-Based Video Retrieval (CBVR) search engine based on the LIRE project . In other words, LIvRE makes it possible to efficiently search through small and huge video collections using images as input queries.

Features Overview

  • Open source.
  • Sate-of-the-art features based on the LIRE CBIR library.
  • Powerful performance.
  • Covers the 3 main blocks of a full CBVR system: Extraction, Indexing and Retrieval.
  • Uses Apache Solr as search server.
  • Scalable.
  • Multi-platform and OS independent.

Requirements

Before proceeding to index your video collection, make sure the system where the video retrieval server will run fulfills the following dependencies. It is recommended to add them to the system path.

  • ffmpeg
  • cURL
  • Python 3.x
  • Java SE Development Kit 8 (OpenJDK seems to have issues with the Solr server)

Instructions

  1. Download and extract the LIvRE Project files.
  2. Download and extract Apache Solr v4.10.2 (other versions not tested).
  3. Copy the Lire Request Handler from the LIvRE Project files,"lire-request-handler.jar" , at \solr-4.10.2\example\solr-webapp\webapp\WEB-INF\lib
  4. Replace "schema.xml" and "solrconfig.xml" files on \solr-4.10.2\example\solr\collection1\conf with the ones from the LIvRE Project files.
  5. Apache Solr v4.10.2 (other versions not tested)
